## 🚀 快速開始

### 環境需求
- Node.js 16+ 
- npm 或 yarn

### 安裝與執行

1. **安裝依賴**
   ```bash
   npm install
   # 或
   yarn install
   ```

2. **編譯 TypeScript**
   ```bash
   npm run build
   # 或
   yarn build
   ```

3. **開發模式（監聽檔案變化）**
   ```bash
   npm run dev
   # 或
   yarn dev
   ```

4. **啟動本地伺服器**
   ```bash
   npm run serve
   # 或
   yarn serve
   ```

5. **開啟瀏覽器**
   訪問 `http://localhost:3000`
